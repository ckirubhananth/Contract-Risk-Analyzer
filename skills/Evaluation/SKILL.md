# Evaluation Skill

Runs unit, golden-dataset, and LLM-as-judge tests for a skill.

## Structure
- `scripts/`: Executable code implementing the skill logic.
- `references/`: Supporting documentation, papers, or datasets.
- `assets/`: Static templates or helper resources.
- `tests/`: Unit and evaluation tests for verification.
